# 200 Club Website Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a single-file static HTML splash page for The 200 Club of Hawaii with navy/gold branding, HPD and HFD badges, mission statement, how-it-works steps, and an embedded Zeffy donation form.

**Architecture:** All HTML, CSS, and content lives in one file — `200-club/index.html`. No build step, no frameworks, no npm. CSS is written inline in a `<style>` block. The Zeffy donation form is an `<iframe>` embed; a styled placeholder is used until the Zeffy account is created.

**Tech Stack:** Vanilla HTML5, CSS3. No JavaScript required. Hosted on any static file server (GitHub Pages, Netlify, etc.).

## Replacing Placeholders Before Launch

These steps are not part of the build — they happen when real assets are ready:

**Mission text:** Open `200-club/index.html`, find the `<!-- MISSION -->` section, replace the `<p>` body text with the exact bylaws language.

**Zeffy embed:** Follow the instructions in the `<!-- ZEFFY EMBED -->` comment block. Replace the placeholder `<div>` with the `<iframe>` tag from your Zeffy dashboard.

**Badge images:** Replace the two `src` attributes in the `<!-- BADGES -->` section with official HPD and HFD logo URLs (or paths to locally saved files in `200-club/assets/`).
